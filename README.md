# fbmonitor_example
一個Line輿情通知機器人，會自動去抓粉絲頁貼文，用Line bot api來丟訊息

## 開發環境
Python + Google Appengine + Flask

- app.yaml 要填appengine id
- fbcrawler 要填申請的facebook app資訊
- linebot 要填申請的Line Bot API資訊
